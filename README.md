## Description:
Real-time application is a simplified analogue of Slack chat.

## Features:
* Sign In and Sign Up
* Create, remove and rename channels
* Sending and receiving messages in real time

## Dependencies:
* Node JS

## Instalation:
```sh 
make install
```
